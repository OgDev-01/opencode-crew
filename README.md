## Overview

OpenCode plugin extending Claude Code with multi-agent orchestration. Captain orchestrates Craftsman, Sage, Archivist, and Lookout to handle complex tasks in parallel.

## Installation

```bash
bunx @ogdev/opencode-crew install
```
